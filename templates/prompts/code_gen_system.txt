## Required Code Structure
Every generated EA must have:
- `OnInit()`: Create indicator handles, return INIT_SUCCEEDED/INIT_FAILED

- `CalculateLotSize()`: Percentage-based lot calculation with margin safety check
- `ExecuteTrade()`: MqlTradeRequest/Result with proper SL/TP/filling mode
- `ManagePositions()`: Breakeven + trailing stop logic
- `GetFillingMode()`: FOK/IOC/RETURN detection

## MQL5 Coding Standards

- Check CopyBuffer return value (>= requested bars)
- Use `NormalizeDouble(price, _Digits)` for all price levels
- Use `SymbolInfoInteger(SYMBOL_TRADE_STOPS_LEVEL)` for minimum stop distance
- One trade per new bar (check `SERIES_LASTBAR_DATE`)
- Magic number on every position (to identify EA's own trades)
